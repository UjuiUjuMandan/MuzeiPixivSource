# MuzeiPixivSource
[Muzei] 的 [Pixiv] 圖片來源

一個定時自動抓取[Pixiv]上的插畫並設定為桌布的小程式

## 功能
* 定時更換
* 下載原始解析度圖片
* 支援的模式
  * 推薦
  * 收藏
  * 排行榜
* 過濾
  * 最少瀏覽次數
  * 最少收藏次數
  * R18 圖片
  * 尺寸過小的圖片

## 使用方法
1. 安裝 [Muzei]
2. 安裝 [Muzei Pixiv Source]
3. 開啟[Muzei Pixiv Source]並登入 [Pixiv] (非必須，建議)
4. 從[Muzei]中設定此應用為圖片來源
5. 享受好看的桌布！
